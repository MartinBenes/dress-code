--- name: Adding a tooltip title: Adding tooltip notes: | Apply the class ```dc--has-tooltip``` class and a positioning class (ex. ```dc--has-tooltip--left```) to any element show a tooltip when hovering. The attribute ```data--dc-has-tooltip``` must be present and not empty: the value will be used as the content for the tooltip body. The tooltip implementation is based significantly on [hint.css](http://kushagragour.in/lab/hint/). ---
