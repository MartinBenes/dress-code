--- name: Icons title: Icons notes: | Together with the icon font, Dress Code distribute also the corresponding svg files. Those svg files are used in some components. Dress-code assumes that the svg files will be located in the ../img/svg-icons/ directory, relative to the compiled CSS files. Moving or renaming the image folder means updating the CSS in one of two ways: - Change the $dc-image-path variable in the source SASS files. - Change the url() path in the compiled CSS. Use whatever option best suits your specific development setup. ---
