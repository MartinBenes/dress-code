--- notes: | Variables used in Dress Code Grid and their default values ---
Variable Type Default Description
$dc-grid-column-count Number 12 Number of total columns in the grid.
$dc-grid-gutter Map (small: 16px, medium: 24px, large: 32px) Gutter size at different breakpoints.
$dc-grid-breakpoints Map (small: 20em, medium: 37.5em, large: 64em, huge: 80em) Breakpoint sizes.
The names of column classes are based on these breakpoint names.
This bloats the generated CSS so use only what you need.
