--- notes: | Want to work with SCSS mixins directly, here are some details: ---

dc-grid-row

Mixin for creating a row.
Sets the margin for row based on the gutter space per breakpoint.

Parameter Type Default Description
$gutter Map $dc-grid-gutter A Map of gutter values for different breakpoint sizes. See default values under Variables (below).

dc-grid-column

Mixin for creating a column.
Sets the flex value for the column
Sets the padding for column based on the gutter space per breakpoint.

Parameter Type Default Description
$columns null | Number null Size (or span) of the column.
$gutter Map $dc-grid-gutter A Map of gutter values for different breakpoint sizes. See default values under Variables (below).

dc-block-grid

Mixin for creating a block grid.
Sets the flex value for the child columns

Parameter Type Default Description
$columns Number Number of columns in a row.
$column-selector String ".dc-column" The selector class for columns within the row

grid-column-flex

Function for calculating flex property value for a column.
Returns flex property value based on the number of columns supplied.

Parameter Type Default Description
$columns null | Number null Size (or span) of the column.

grid-column-size

Function for calculating the width of a column in percentage.
Returns percentage value of a column in relation to the total number of columns

Parameter Type Default Description
$columns Number Size (or span) of the column.
